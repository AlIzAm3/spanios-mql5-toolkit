# spanios-mql5-toolkit
A lightweight open-source toolkit for building professional MetaTrader 5 Expert Advisors.
Overview

Spanios MQL5 Toolkit is a collection of reusable components designed to simplify the development of professional Expert Advisors (EAs) for MetaTrader 5.

The project focuses on providing common infrastructure that many trading systems require, allowing developers to spend more time on strategy development instead of rebuilding the same utilities for every project.

This toolkit does not include any trading strategy or signal generation logic. It only provides general-purpose components that can be used in any MQL5 project.

Features

Current

- Project structure
- Documentation

Planned

- Logging system
- Configuration manager
- Risk management utilities
- Session and market time utilities
- Symbol information helpers
- File utilities
- HTTP request wrapper
- JSON utilities
- Notification helpers
- Reusable examples

Project Goals

- Reduce boilerplate code in MQL5 projects.
- Provide reusable and well-documented components.
- Encourage clean and modular architecture.
- Make Expert Advisor development easier for beginners and professionals.
